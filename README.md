## 프로젝트 설명

이 프로젝트는 다양한 스케줄 관리 기능을 제공하는 리액트 기반 애플리케이션입니다. 다음과 같은 용도로 사용할 수 있습니다:
- **일일 계획**: 개인의 일일 일정을 관리할 수 있는 기능
- **과목 시간표**: 학생을 위한 과목 시간표 생성 및 관리
- **환자 차트**: 병원에서 환자의 진료 일정을 기록하고 관리

## 구성 요소

### 구성 버전 및 라이브러리
- Node.js 14.x 이상
- react 18.x 버전
- bundler vite
- redux
